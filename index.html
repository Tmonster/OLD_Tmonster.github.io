Intrusion Detection System

One assignment for my Computer Security and Privacy Course was to build an Intrusion Detection System. The IDS was built to monitor network traffic by inspecting packets and identifying attacks based on provided signatures in the assignment. Internet traffic was saved using the tcpdump command and then parsed from the pcap files produced from the Linux command. If a attack signature was detected, then program would print an alert with the details of the attack. The following attacks were identified

  • Spoofed packetd
  • Unauthorized servers
  • Sinkhole lookups
  • ARP spooding
  • IIS worms
  • NTP reflection DDoS
